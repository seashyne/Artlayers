\# Project Rules



\## Coding Style

\- Use TypeScript strict mode

\- Prefer functional programming

\- Keep files under 300 lines

\- Avoid prop drilling

\- Prefer composition over inheritance



\## Architecture

\- UI and rendering must stay separated

\- Rendering logic must never be inside React components

\- Business logic goes into engine/



\## Performance

\- Avoid unnecessary rerenders

\- Use memoization carefully

\- Batch rendering updates



\## Code Quality

\- No dead code

\- No console.log in production

\- Add comments for complex rendering logic



\## UI Direction

Modern minimal dark UI inspired by:

\- Figma

\- Linear

\- Procreate
